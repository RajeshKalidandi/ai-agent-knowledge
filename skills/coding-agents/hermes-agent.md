# hermes-agent

Open-source AI agent framework by Nous Research — multi-platform, self-improving, extensible

## Installation

```bash
curl -fsSL https://raw.githubusercontent.com/NousResearch/hermes-agent/main/scripts/install.sh | bash
```

## Key Commands

| Command | Description |
|---------|-------------|
| `hermes` | Start interactive chat session |
| `hermes setup` | Run interactive setup wizard |
| `hermes doctor` | Check system health and dependencies |
| `hermes -q 'query'` | Single non-interactive query |
| `hermes cron create '0 9 * * *'` | Schedule a cron job |
| `hermes webhook subscribe name` | Create webhook subscription |
